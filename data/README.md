# Data Sources

This folder contains the raw datasets used in the project. The files have been renamed into clean, GitHub-friendly names while preserving the original data content.

## Raw datasets included

| Project filename | Original upload name | Source / publisher | Notes |
|---|---|---|---|
| `cost_of_living_index_2022.csv` | `Cost_of_Living_Index_2022.csv` | Numbeo | Country-level 2022 cost-of-living index and component indices. |
| `spareroom_rental_listings_2023_04_16.csv` | `Spareroom2023-04-16.csv` | SpareRoom / rental listings extract | UK room/rental listing data captured on 16 April 2023. |
| `headline_inflation_index.csv` | `inflation_data_dataset.csv` | IMF Data / macroeconomic inflation dataset | Headline consumer price inflation by country and year. |
| `gdp_index.csv` | `GDPindex.csv` | IMF Data / macroeconomic index dataset | GDP deflator index growth-rate data by country and year. |
| `food_price_index.csv` | `datasetfoodindex.csv` | IMF Data / macroeconomic inflation dataset | Food consumer price inflation by country and year. |
| `producer_price_index.csv` | `ProducePriceIndexDataset.csv` | IMF Data / producer price dataset | Producer price inflation by country and year. |
| `average_house_prices_uk.csv` | `avgmortgageratesUK - Average price.csv` | UK House Price Index / HM Land Registry | Regional average house price data. |
| `housing_price_index_uk.csv` | `housingindexuk.csv` | UK House Price Index / HM Land Registry | Regional house-price index data. |
| `property_sales_volume_uk.csv` | `salesvolumeUk.csv` | UK House Price Index / HM Land Registry | Regional property sales volume data. |
| `spareroom_gdp_index.xlsx` | `spareroom statistics UK - GDPindex.xlsx` | SpareRoom / supporting GDP sheet | Supporting Excel data used in the original dissertation workflow. |
| `spareroom_gdp_index_secondary.xlsx` | `spareroom statistics UK - GDPindex-2.xlsx` | SpareRoom / supporting GDP sheet | Secondary copy/version of the supporting Excel data. |
| `consumer_price_inflation_reference_tables.xlsx` | `consumerpriceinflationdetailedreferencetables.xlsx` | Office for National Statistics (ONS) | Detailed UK CPI, CPIH, and RPI reference tables. |

## Notes

- `data/raw/` contains original raw inputs.
- `data/processed/` is generated by running `python src/data_preprocessing.py`.
- The raw data is included in this zip for portfolio reproducibility.
- Before publishing publicly, check each source's reuse/licensing terms and cite the source pages in your GitHub README.
